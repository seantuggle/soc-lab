How it works:
Every event ingested is checked against the IOC database automatically. If a src_ip matches a known-bad IP, or a dns_query matches a known C2 domain, the event's fields are tagged with src_ip_ti_verdict, src_ip_ti_tags, and src_ip_ti_score. These appear in the investigation view under Threat Intelligence.

Feeds: Click ↻ Fetch to pull a feed now. Large feeds (Feodo, ET) may take a few seconds. IOCs from feeds are stored locally so lookups are always instant — no API call on every event.

API keys: Set ABUSEIPDB_API_KEY and/or VIRUSTOTAL_API_KEY in your docker-compose.yml environment block for live reputation lookups on IPs not in your local IOC table. Results are cached for 24 hours.
